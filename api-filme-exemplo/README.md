## API DE FILMES

## DESCRIÇÃO
O desenvolvimento dessa API tem como objetivo criar uma aplicação que seja capas de cadastrar e gerenciar filmes.

### ✔️ TÉCNICAS E TECNOLOGIAS UTILIZADAS
- ASP.NET Core
- SGBD (PostGresSQL)
- Paginação


### O QUE FOI DESENVOLVIDO
- CRUD - FILMES
